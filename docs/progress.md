# Amber 项目进度

> 最后更新：2026-06-08

本文记录 Amber v1 已完成的功能和已知差距，作为后续迭代的基准。

---

## v1 已完成

### 核心管线

| 功能 | CLI 命令 | 状态 |
|---|---|---|
| 抓取网页（静态 + 动态站点） | `amber import <url>` | ✅ |
| 列出已导入内容 | `amber list` | ✅ |
| 删除记录 | `amber delete <id>` | ✅ |
| 重新抓取 | `amber reimport <id>` | ✅ |
| 标签管理 | `amber tag ls/add/rm` | ✅ |
| 文件→PG 数据迁移 | `amber migrate` | ✅ |
| 浏览器诊断安装 | `amber doctor` | ✅ |

### Web 阅读界面

| 功能 | 说明 | 状态 |
|---|---|---|
| 文章列表 | 标题 + 来源 + 时间 + 摘要 | ✅ |
| 文章阅读 | Markdown 渲染 + 图片 | ✅ |
| 分栏阅读 | 支持双栏排版 | ✅ |
| 专注阅读目录 | 侧边目录导航 | ✅ |
| 视频播放 | 内嵌视频支持 | ✅ |
| 阅读进度 | 已读/未读标记 | ✅ |
| 标签管理 | Web 端标签增删 | ✅ |
| 主题切换 | 明/暗主题 | ✅ |

### Web 服务管理

| 子命令 | 说明 | 状态 |
|---|---|---|
| `amber web serve` | 启动 Web UI（后台守护进程） | ✅ |
| `amber web status` | 查看服务运行状态 | ✅ |
| `amber web stop` | 停止服务 | ✅ |
| `amber web logs` | 查看服务日志 | ✅ |

### 存储后端

| 模式 | 触发条件 | 状态 |
|---|---|---|
| 文件存储（零配置） | 不设环境变量 | ✅ |
| PostgreSQL | 设 `DATABASE_URL` | ✅ |
| Cloudflare R2 | 设 `R2_*`（4 个） | ✅ |
